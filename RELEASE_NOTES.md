# 🚀 Release v5.0.4 "The Pollinations Multi-Tool"

## Внедрение экосистемы Pollinations AI

Мы полностью переработали архитектуру и подключили мощнейший стек моделей от Pollinations, превратив бота в настоящий швейцарский нож для работы с контентом.

## 🚀 Главное: Внедрение экосистемы Pollinations AI

*   **💬 Smart Chat (LLM):** Теперь в чате доступны топовые модели (OpenAI, Gemini, Claude) через единый шлюз. Ответы стали быстрее, а логика — острее.
*   **🎨 Next-Gen T2I (Text-to-Image):** Задействована модель Flux Schnell. Генерация картинок по описанию (промпту) вышла на новый уровень детализации и фотореализма.
*   **👁 AI Vision:** Бот теперь «видит» ещё и через сервис Pollinations! Благодаря интеграции Vision-моделей, теперь можно отправлять фото и получать его детальный разбор, описание или перевод текста с изображения подробно и на русском языке.
*   **🎙 STT (Speech-to-Text):** Внедрена высокоточная транскрипция аудио через Whisper Large V3. Голосовухи больше не проблема — бот переводит их в текст за секунды.

## 🛠 Технические улучшения

*   **Hybrid API Connectivity:** Оптимизирована работа с заголовками и авторизацией для стабильного соединения с серверами генерации.
*   **Adaptive Image Sizing:** Система теперь динамически подбирает идеальное разрешение (Size) под конкретную модель, исключая ошибки 400 (Bad Request).
*   **Clean Data Pipeline:** Очистка промптов от лишнего мусора и автоматический перевод на лету для лучшего понимания нейронками ваших идей.

## 🛡 Стабильность и Баланс

*   **Auto-Refill System:** Использование системы "самовозобновляемых" кредитов Pollinations позволяет генерировать контент без жестких лимитов.
*   **Debug Master:** Расширенное логирование всех этапов запроса (от KV до API) для мгновенного исправления любых затыков.

---

## 🚀 Ключевое изменение v5.X: Переезд на Yandex.Cloud
- **Среда выполнения**: Бот теперь работает в полноценной среде Node.js 22, что обеспечивает стабильность и расширяемость.
- **База данных**: В качестве основного хранилища данных (вместо Cloudflare KV) используется **Yandex Database (YDB)**.
- **Архитектура**: Разработан специальный адаптер (`index.js` + `db_adapter.js`), который эмулирует среду Cloudflare Workers, позволяя сохранить и адаптировать основную кодовую базу `worker.js`.
