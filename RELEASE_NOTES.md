# 🤖 Release Notes: Gemini Bot v5.0.5 (01.04.2026) — "Deep Context & S3 Power"

## 🧠 Единый интеллект и S3

-   **Бесконечная память (S3 Integration):** Бот больше не забывает контекст при перезагрузке сервера. Вся история ваших диалогов теперь мгновенно сохраняется и подгружается из облачного хранилища S3.
-   **Мультиплатформенный контекст:** Начни общение с Джемини в ТГ, и если завтра ты зайдешь в него через другой апп или сервис нашей экосистемы — он вспомнит всё. История едина для твоего ID, где бы ты ни находился.

## ⚡️ Авторизация и Mini App

-   **Seamless Login:** Внедрена технология бесшовного входа для Telegram Mini App. Приложение внутри бота теперь открывается мгновенно, автоматически подтягивая твой профиль через наш новый Auth-шлюз.
-   **Smart Data Sync:** Исправлена синхронизация данных профиля. Имя, фамилия и юзерпик теперь всегда актуальны и подгружаются напрямую из Telegram.

## 🛠 Технические правки (Under the Hood)

-   **API Gateway Optimization:** Полностью переписана логика взаимодействия бота с бэкендом на Yandex Cloud. Устранены ошибки редиректов и "мертвые" петли при входе.
-   **History Recovery:** Добавлена система автоматического восстановления сессии. Если соединение прервалось, бот подтянет последние сообщения из S3, чтобы диалог не прерывался.

---

## 🚀 Ключевое изменение v5.X: Переезд на Yandex.Cloud
- **Среда выполнения**: Бот теперь работает в полноценной среде Node.js 22, что обеспечивает стабильность и расширяемость.
- **База данных**: В качестве основного хранилища данных (вместо Cloudflare KV) используется **Yandex Database (YDB)**.
- **Архитектура**: Разработан специальный адаптер (`index.js` + `db_adapter.js`), который эмулирует среду Cloudflare Workers, позволяя сохранить и адаптировать основную кодовую базу `worker.js`.
