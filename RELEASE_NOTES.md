# 🚀 Релиз версии 4.6 | 11.03.2026

Это обновление включает в себя значительные улучшения в работе с моделями Gemini, а также расширенные возможности по обработке видео и анимации.

---

### ✨ **Что нового:**

*   **Расширенная поддержка моделей:**
    *   Добавлена поддержка моделей `gemini-2.5-flash-lite` и `gemini-2.5-flash`.
    *   Интегрирована модель `Kling 2.6` для преобразования видео в видео с управлением движением.
    *   Заменена модель `Grok T2I` на `google nano-banana`.

*   **Улучшенная обработка видео и анимации:**
    *   Появилась новая функция для преобразования видео в GIF.
    *   Добавлена возможность создания анимированных GIF-стикеров.
    *   Улучшена обработка GIF и стикеров.

---

### 📈 **Улучшения:**

*   **AI-Прокси:**
    *   Реализован полный механизм резервирования для прокси Gemini.
    *   Добавлена ​​видимость ошибок и отладка для прокси-запросов.
    *   Включен текстовый чат с Gemini через прокси.
    *   Маршрутизация T2I и I2I через AI-прокси.

*   **Обработка видео:**
    *   Изменен режим обработки Kling на 720p.
    *   Исправлена ​​ошибка `endVal` при обработке видео.

---

### 🛠️ **Исправления:**

*   Исправлена ​​ошибка обработки стикеров.
*   Устранена проблема с `callbackQueryId`.

---

### ⚙️ **Технические изменения:**

*   **Реализация AI-прокси:**
    *   Реализован прокси с использованием привязок.
    *   Добавлены проверки переменных окружения.

*   **Переключение моделей:**
    *   Переключена модель Gemini.
    *   Заменена модель V2V на `kling-2.6/motion-control`.

*   **Очистка кода:**
    *   Удалены ненужные данные из хранилища KV.
    *   Удален ключ `_last_video` из KV.

*   **Развертывание:**
    *   Внесены изменения в файл `deploy.yml`.

*   **Другое:**
    *   Добавлено `Date.now()` к именам файлов.
    *   Обновлен `README.md`.
