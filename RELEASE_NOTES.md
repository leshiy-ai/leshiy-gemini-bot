# 🚀 Release v5.0.2 | 27.03.2026: Победа над аудио-данными!

Этот релиз знаменует собой полную поддержку аудио-сообщений и представляет значительные улучшения в их обработке.

## ✨ Новые возможности и улучшения:

*   **Полная поддержка аудио:** Теперь бот может полноценно работать с аудио-сообщениями, включая их получение, обработку, отправку, распознавание и генерацию.
*   **Оптимизация аудио-потоков:** Улучшена производительность при работе с аудио-данными, что снижает задержки и повышает стабильность.
*   **Повышенное качество распознавания:** Внедрены новые алгоритмы для более точного и быстрого распознавания речи.

## 🐞 Исправленные ошибки и улучшения:

    *   **Binary Integrity Fix:** Решена критическая проблема искажения бинарных данных (UTF-8 Replacement Character EF BF BD). Теперь MP3-поток передается без потерь.

    *   **TTS Multi-Engine Support:** Полная поддержка и отладка пяти независимых движков:
    *   **Workers AI (@cf/deepgram)** — внедрен прямой fetch в обход текстовых прокси для максимальной скорости.
    *   **Gemini (2.5-flash-preview-tts)** — стабильная работа через JSON-контейнер и локальную конвертацию PCM → MP3.
    *   **Kie.ai (ElevenLabs)** — исправлена ошибка передачи chatId. Асинхронные колбэки теперь прилетают точно в цель.
    *   **Bothub (OpenAI tts-1)** — корректная обработка application/octet-stream.
    *   **VoiceRSS** — классическая озвучка без сбоев.

    *   **Architecture Optimization:**
    *   Исправлено ветвление в основном цикле `say_run`: теперь система различает синхронные (Workers AI/Gemini) и асинхронные (Kie.ai) модели, не вызывая ошибок при пустом `audioBase64`.
    *   Усилен дебаг-режим: теперь в логи выводится заголовок Base64 (Head), что позволяет мгновенно диагностировать целостность файла.

## 🚀 Ключевое изменение v5.X: Переезд на Yandex.Cloud
- **Среда выполнения**: Бот теперь работает в полноценной среде Node.js 22, что обеспечивает стабильность и расширяемость.
- **База данных**: В качестве основного хранилища данных (вместо Cloudflare KV) используется **Yandex Database (YDB)**.
- **Архитектура**: Разработан специальный адаптер (`index.js` + `db_adapter.js`), который эмулирует среду Cloudflare Workers, позволяя сохранить и адаптировать основную кодовую базу `worker.js`.
